# Переход на облачное хранилище данных

## Рекомендуемое решение: Supabase (PostgreSQL + Realtime)

### Преимущества:
- ✅ Real-time синхронизация (видите обновления мгновенно)
- ✅ PostgreSQL база данных (надёжна, мощна)
- ✅ Встроенная REST API
- ✅ Бесплатный уровень достаточен для ресторана
- ✅ WebSocket для live обновлений

---

## Шаг 1: Создать учётную запись на Supabase

1. Перейти на https://supabase.com
2. Зарегистрироваться через GitHub/Google
3. Создать новый проект (выбрать регион ближе к вашему)
4. Сохранить:
   - Project URL
   - API Key (anon/public)
   - Database password

---

## Шаг 2: Установить Python библиотеки

```bash
pip install supabase python-dotenv websocket-client
```

---

## Шаг 3: Создать файл конфигурации

Создать `.env` в корне проекта:
```
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your-anon-key
```

---

## Архитектура облачного решения

```
┌─────────────────────────────────────────────────────────────┐
│                      SUPABASE (Cloud)                       │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  PostgreSQL Database                                  │  │
│  │  ├─ orders                                            │  │
│  │  ├─ menu_availability                                │  │
│  │  ├─ purchase_orders                                  │  │
│  │  ├─ suppliers                                        │  │
│  │  └─ products                                         │  │
│  └───────────────────────────────────────────────────────┘  │
│                          ↕ Real-time                         │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  REST API + WebSocket                                │  │
│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
         ↕
    ┌────────────────────────────────────────┐
    │   Ваш локальный сервер (ресторан)     │
    │   dragon_local_server.py (обновлено)     │
    │   - Синхронизирует заказы              │
    │   - Отправляет данные в облако        │
    └────────────────────────────────────────┘
         ↕
    ┌────────────────────────────────────────┐
    │   Веб-интерфейс (дома)                 │
    │   - Видит все заказы в real-time       │
    │   - Управляет меню доступностью       │
    │   - Смотрит статистику                 │
    └────────────────────────────────────────┘
```
